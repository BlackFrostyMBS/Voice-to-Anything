# Voice to Anything

Real-time voice dictation tool — speak and your words are typed into any active window. Powered by OpenAI Whisper. Supports 99 languages.

## Install

1. Double-click `install.bat`
2. A shortcut called **Voice to Anything** will appear on your Desktop
3. Done

## Usage

Double-click the desktop shortcut to launch, then:

| Key | Action |
|-----|--------|
| `F9` | Manual mode — press to start recording, press again to stop and type |
| `F8` | Auto mode — always listening, types automatically when you pause speaking |
| `ESC` | Quit |

**Tip:** Click inside the window where you want the text to appear before speaking.

## Supported Languages

Whisper auto-detects the language you speak — English, Hebrew, Russian, Arabic, and 95 more. No setup needed.

## Requirements

- Windows 10 / 11
- Python 3.11 (installed automatically by `install.bat`)
- FFmpeg (installed automatically by `install.bat`)
- A microphone
